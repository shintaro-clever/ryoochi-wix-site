# Skills Runtime Contract

- 実行ディレクトリは常に repo root とする。
- 実行物は相対パスを前提に実行する。
- 生成物の出力先は `tmp/` または `reports/` のみとする。
- `tmp/` は一時生成物、`reports/` は成果レポートを配置する。
- Secrets（APIキー/トークン）は出力しない。
- `.env` の内容を表示しない。
- `auth.json` の内容を表示しない。
- 失敗時は失敗コマンドをそのまま記録する。
- 失敗時は stderr 末尾をそのまま記録する。
- 推測や要約で失敗原因を書き換えない。
