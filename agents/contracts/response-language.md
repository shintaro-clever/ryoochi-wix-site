# Response Language Contract

この契約は全エージェント/コマンドの返答言語を定義する。
返答本文は常に日本語で出力する。
思考の有無は任意だが、返答本文は日本語固定とする。
ユーザーが英語での返答を明示した場合のみ英語を許容する。
英語指定がない限り、英語で返さない。
コードブロックは原文のまま出力してよい。
エラーログ/スタックトレースは原文のまま出力してよい。
APIレスポンスなど機械可読な出力も原文のまま可。
翻訳で意味が壊れる箇所は翻訳しない。
誤って英語で返答した場合、以後は日本語で出し直す。
同一メッセージ内で言語を混在させない（例外はコード/ログのみ）。
日本語出力を最優先し、可読性を重視する。
この契約は他のルールより優先される。
