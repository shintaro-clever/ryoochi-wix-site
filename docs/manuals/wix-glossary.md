# Wix Glossary

## Wix Studio
- サイトの見た目や配置を調整する作業画面です

## 案件リポジトリ
- この案件専用の資料や原型を置く保管場所です

## 共通土台
- 複数案件で共通に使うベースの考え方や仕組みです

## 最小検証ページ
- いきなり本番全体を作らず、まず小さく試すためのページやセクションです

## AI制作物
- AI が作った原型、文言案、構造案、たたき台のことです

## 静的原型
- HTML と CSS で作った、持ち込み元になる見本ページです

## 管理者確認
- 自分だけで判断せず、管理者や技術担当に相談して進めることです

## 非エンジニア
- コード修正を前提にせず、運用や調整を担当する人です

## Git
- 変更履歴を管理するための仕組みです

## CLI
- 画面ではなく、文字入力で使う操作方法です

## 接続
- Wix Studio や案件資料を、使える状態に結びつけることです

## 編集性
- 後から文言、画像、配置を直しやすいかどうかです

## 再編集性
- 一度作った後でも、あとから無理なく直し続けられるかどうかです

## CTA
- クリックしてほしい案内やボタンのことです

## 持ち込み
- repo 側の原型や案を、Wix Studio 側で再構成して使うことです

## 引継ぎ
- 次の担当者が迷わず進められるように、状況や判断材料を渡すことです
